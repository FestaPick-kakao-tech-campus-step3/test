---

## 2) 소개
FestaPick은 **축제 참여자**와 **축제 관리자(주최측)** 모두가 함께 쓰는 플랫폼입니다.  
- 참여자: 관심 지역·스타일 기반 추천, 실시간 공지/채팅, 위시·리뷰  
- 주최자: 축제 등록/관리, 공지/메시지 발송, 권한 기반 운영

## 4) 기술 스택
- **Backend**: Spring Boot 3.x, Java 17/21, JPA/Hibernate, QueryDSL  
- **DB/Cache**: MySQL (RDS), Redis (ElastiCache)  
- **Infra**: AWS EC2, ECR, S3, Route53, Nginx, Docker, GitHub Actions (Blue/Green)  
- **Observability**
  - Logs: Elasticsearch, Logstash, Kibana (ELK), Filebeat  
  - Metrics: Prometheus, Spring Boot Actuator  
  - Dashboards: Grafana
